# User Guides

This directory contains user-facing guides and tutorials for the Half Sword AI Agent.

## Available Guides

- **[QUICK_START.md](./QUICK_START.md)**: Quick start guide for new users
- **[HALF_SWORD_CONTROLS.md](./HALF_SWORD_CONTROLS.md)**: Complete control reference
- **[INTERCEPTION_INSTALL.md](./INTERCEPTION_INSTALL.md)**: Interception driver installation guide
- **[DATASET_GUIDE.md](./DATASET_GUIDE.md)**: Dataset building and management guide
- **Rewards for Half Sword Bot Learning.txt**: Reward system documentation

## Getting Started

1. Read [QUICK_START.md](./QUICK_START.md) for installation and basic usage
2. Review [HALF_SWORD_CONTROLS.md](./HALF_SWORD_CONTROLS.md) to understand game controls
3. Install Interception driver using [INTERCEPTION_INSTALL.md](./INTERCEPTION_INSTALL.md) for kernel-level input control
4. Build datasets using [DATASET_GUIDE.md](./DATASET_GUIDE.md) if needed
