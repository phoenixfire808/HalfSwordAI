# Test Files

This directory contains test and utility scripts.

## Test Scripts

- **test_system.py** - System integration tests
- **test_input_injection.py** - Input injection tests
- **test_kill_switch.py** - Kill switch functionality tests
- **test_kill_switch_simple.py** - Simplified kill switch test
- **test_llm_debugging.py** - LLM integration debugging

## Utility Scripts

- **setup_ollama.py** - Ollama setup and verification
- **verify_integration.py** - Integration verification
- **view_performance.py** - Performance visualization
- **safety_check.py** - Safety system checks
- **alignment_check.py** - System alignment verification

## Running Tests

```bash
python tests/test_system.py
python tests/verify_integration.py
```
